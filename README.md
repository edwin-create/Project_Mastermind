
An interactive implementation of the classic code-breaking board game, Mastermind.

## 🎮 About the Game
Mastermind involves a codemaker setting a secret four-color peg code (with duplicates allowed). A codebreaker then attempts to guess the code within a set number of turns. Feedback is given after each guess: a black peg for a correct color in the right place, and a white peg for a correct color in the wrong place. The codebreaker wins by correctly guessing the code.

## 🚀 Key Features
*   **Command-line Interface (CLI)**: Playable via the console.
*   **Random Code Generation**: Creates a unique secret code each game.
*   **Feedback System**: Provides feedback based on black/white peg logic.
*   **Limited Attempts**: Offers up to 8 guesses to break the code.
